# ADE Denial Letter Customization Prototype

Static, self-contained proof of concept for certification staff denial letter customization.

## Open

Open `index.html` in a browser. No backend, install step, or build step is required.

## Prototype Scope

- Certification staff view only.
- Top-level educator dropdown for switching between sample cases.
- Editable letter date and credential fields.
- Configurable statute descriptions with statute reference codes, pre-canned statute denial reason options, educator-facing support language, and evaluator notes.
- Custom manual denial reason creation.
- Manually editable denial letter preview with focused Save Draft, Reset Demo, and Preview Letter actions.
- Local browser storage for demo persistence, with a reset button.
